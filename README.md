# Snake Game

A hands-on guide to **agentic coding with GitHub Copilot**, built around a minimal snake game (HTML5 Canvas + JavaScript). Use arrow keys or WASD to control the snake.

## How to run

Open `index.html` in your browser.

## Exercise

This game is the starting point for an agentic coding exercise. Follow the guided tour (auto-opens in VS Code) or read [STEPS.md](STEPS.md) for the full walkthrough.
